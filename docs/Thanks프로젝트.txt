1. 유기견 : 
포인핸드는 임시보호소
목표
- 입양은 입양후 방문이 적다 어떻게 방문을 더 하게 할 것이냐?
- 홈페이지를 지속적으로 방문하게 할 수 있는 방법? 
- 예약을 잡아서 my page에 데이터가 쌓이도록 구현
- 목적 : 모든 사람들이 우리 사이트를 통해서 입양이 가능하도록 하는 것 

기능
- 쇼핑몰에 있는 api로 상품정보 당겨올 수 있다. 이걸 당겨와서 결재가 가능하도록 할 것 인지 흉내만 낼 것인지
- 카드사 데이터 넘기는건 안댐, 결제 대행 서비스 사용하려면 -> 보안 서비스 필요
- 네이버 지도 open api 유  
- 쇼핑몰 :  
- 핵심기능을 우선순위로 나누어서 우선순위로 해결해 나가는 방식
